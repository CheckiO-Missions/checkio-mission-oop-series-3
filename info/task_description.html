Mission 3: Initializing objects.

3.2. It is usually not necessary, but since you already have my_car object, created without passing any arguments, you now need to add default values to __init__ arguments. Let it be an empty string "".

3.3. Create two new car objects - some_car1, some_car2 - by passing the brand and model values of type str as arguments.
