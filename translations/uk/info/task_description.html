Тобі потрібен спеціальна функція класу для створення нових об'єктів та встановлення їхніх атрибутів. Функція __init__() викликається автоматично щоразу як клас використовується для створення нового об'єкта.

3.1 Додай функцію __init__() до класу Car, який приймає аргументи brand і model і присвоює їх відповідним атрибутам об'єкта.

3.2. В тебе вже є обʼєкт my_car який був створенний без передачи додаткових аргументів. Тож, аби не викликати помилки при створенні обʼєкта таким чином, додамо значення по замовченню для аргументів функції __init__(). Вони повинні мати значення у вигляді порожнього рядка "".

3.3 Створи два нових об'єкти автомобілі - some_car1, some_car2, використовуючи цей метод, передавши конкретні значення brand і model типу рядок (str).
