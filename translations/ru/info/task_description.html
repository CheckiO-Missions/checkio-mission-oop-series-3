3.1. Добавьте в класс Car функцию __init__(), которая принимает аргументы brand и model и присваивает их соответствующим атрибутам объекта.

3.2. У вас уже есть объект my_car который был создан без передачи дополнительных аргументов. Поэтому, чтобы не возникало ошибок при создании объекта таким образом, добавим значение по умолчанию для аргументов функции __init__(). Они должны иметь значения пустой строки "".

3.3. Создайте два новых объекта Car передав в качестве аргументов следующие строковые значения: some_car1: brand - Ford, model - Mustang; some_car2: model - Camaro. Обратите внимание, что второй автомобиль имеет только модель, его марка должна оставаться по умолчанию.
